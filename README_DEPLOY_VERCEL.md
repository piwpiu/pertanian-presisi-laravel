# Konfigurasi Deploy Laravel ke Vercel

Salin file berikut ke root project Laravel:

- `api/index.php`
- `vercel.json`
- `.vercelignore`
- `.gitignore`
- `.env.example`

Lalu ubah file berikut sesuai versi di folder ini:

- `config/services.php`
- `app/Http/Controllers/DashboardController.php`

Setelah itu push ke GitHub dan import repository dari Vercel.
